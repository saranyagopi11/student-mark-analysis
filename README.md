# 🎓 Student Database Dashboard | Power BI

## 📌 Project Overview

The Student Database Dashboard is an interactive Power BI project designed to analyze student academic performance and provide meaningful insights through visual reports. The dashboard helps educators and administrators monitor student records, attendance, marks, and overall performance.

---

## 🚀 Features

- 📊 Student Performance Analysis
- 👨‍🎓 Total Students Overview
- ✅ Pass & ❌ Fail Analysis
- 📈 Subject-wise Marks Analysis
- 📅 Attendance Monitoring
- 📚 Class-wise Student Distribution
- 🔍 Interactive Filters and Slicers
- 📉 Dynamic Charts and KPIs

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Visualizations

---

## 📈 Dashboard Insights

The dashboard provides insights such as:

- Total number of students
- Pass vs Fail percentage
- Average marks
- Attendance statistics
- Subject-wise performance
- Class-wise student distribution
- Student performance trends
